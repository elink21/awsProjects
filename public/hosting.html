Hosting con Firebase y AWS ‚å®

El proyecto final de este conjunto es precisamente el hosting donde esta alojado esta pagina.


Este proyecto esta alojado mediante un hosting de Firebase que es un producto de Google, para esto se inicializo un proyecto en su consola y se configuro mediante el CLI para subir la pagina a produccion.

Tambien se realizaron pruebas con el Hosting de sitios web estaticos de S3, sin embargo debido a que AWS no provee de un certificado SSL para sus sitios (necesario para el funcionamiento de algunas practicas de esta pagina) se decidio utilizar finalmente un host de Google Firebase. El host de AWS aun es accesible desde aqui:
http://projecthostingb.s3-website-us-east-1.amazonaws.com/.
PD: Este host al no ser HTTPS no es completamente funcional, se recomienda permanecer en esta pagina.
