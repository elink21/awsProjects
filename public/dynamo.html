Dynamo DB 🗃️

Dynamo DB es una base de datos no relacional proveida por AWS que permite guardar datos persistentes sin la necesidad de un servidor con gestor de base de datos propios.

Aqui se demuestra su uso con una aplicacion que permite agregar tablas y muestra las creadas anteriormente, la informacion es persistente sin necesidad de una infraestructura backend para esta pagina.
