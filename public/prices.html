Amazon Pricing 🛒

Con todos los servicios presentes en el catalogo de AWS es facil perderse con las condiciones que establecen el precio de los productos, sin embargo dentro de estos servicios existe uno que precisamente ayuda con este enredo.


Se trata de AWS Pricing, que brinda funciones para listar tanto los servicios presentes en AWS como tambien su tipo de uso y operacion, misma info que facilita el calcular presupuesto de consumo.


Esta ejemplo funcional muestra un select para escoger el servicio, y los datos se visualizaran en la parte derecha de acuerdo a compatibilidad.
